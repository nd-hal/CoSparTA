# Demo covariate matrix

A data frame of simulated covariates for the 1000-session synthetic demo
dataset bundled with CoSparTA. Each row corresponds to one session
(user).

## Format

A data frame with 1000 rows and 3 columns:

- session_id:

  Character. Session identifier (e.g., `"S0001"`).

- cov1:

  Numeric. First simulated covariate.

- cov2:

  Numeric. Second simulated covariate.

## Source

Simulated data generated for package demonstration.
